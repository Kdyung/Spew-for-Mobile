Spew(Mobile)

Spew(Mobile) is a project building on the work from the UCSD Spring 2014 game jam collaboration with Janis.

The project was started during September of 2014 using Unity 4.53 (or so)


It takes the Spew game and adds mobile controls and other features, such as a falling object demo game.


6-11-15 Updated project to unity 5.1.
